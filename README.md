# Cheam Village Dental Studios

A modern, animation-rich landing page for **Cheam Village Dental Studios** — a boutique private dental practice in Cheam Village, Surrey.

**Live site:** see the GitHub Pages URL in the repo sidebar.

## Highlights

- Editorial design system: Fraunces serif + Montserrat, porcelain/ink/sage palette, recreated line-art tooth mark as inline SVG
- GSAP + ScrollTrigger + Lenis smooth scrolling
- Preloader with stroke-drawn logo and staggered hero entrance
- Hero treatment slider with synced headline word rotation and progress tabs
- Scroll-driven pinned hero expansion — the arch image blooms to full-bleed with an overlaid statement (inspired by award-winning practice sites)
- Cursor parallax, magnetic buttons, cursor-following service imagery
- Scroll-scrubbed word reveals, animated counters, auto-rotating testimonials
- Fully responsive, `prefers-reduced-motion` respected

## Run locally

It's a single static `index.html` — open it directly or serve the folder:

```sh
npx serve .
```
